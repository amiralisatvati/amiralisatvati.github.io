## مقدمه

ضرایب همبستگی اسپیرمن و کندال روش‌های **غیرپارامتری** برای سنجش ارتباط بین دو متغیر هستند. برخلاف ضریب پیرسون، این روش‌ها بر اساس **رتبه‌ها** (ranks) محاسبه می‌شوند.

## ۱. ضریب همبستگی اسپیرمن (Spearman's rho)

### تعریف

ضریب اسپیرمن رابطه یکنواختی بین دو متغیر را سنجش می‌کند.

### فرمول

$$\rho = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

جایی که:
- **d_i**: تفاوت رتبه‌های متغیر i
- **n**: تعداد مشاهدات

### نمونه محاسبه

| X | Y | رتبه X | رتبه Y | d | d² |
|---|---|--------|--------|---|----|
| 10| 20| 1      | 1      | 0 | 0  |
| 20| 15| 2      | 2      | 0 | 0  |
| 30| 25| 3      | 3      | 0 | 0  |
| 40| 10| 4      | 4      | 0 | 0  |
| 50| 30| 5      | 5      | 0 | 0  |

### پیاده‌سازی در R

```r
# داده‌های نمونه
X <- c(10, 20, 30, 40, 50)
Y <- c(20, 15, 25, 10, 30)

# محاسبه ضریب اسپیرمن
cor.test(X, Y, method = "spearman")

# نتیجه
# Spearman's rank correlation rho = 0.5
# p-value = 0.3
```

## ۲. ضریب همبستگی کندال (Kendall's tau)

### تعریف

ضریب کندال بر اساس **اتفاقات و عدم اتفاقات** (concordant و discordant pairs) محاسبه می‌شود.

### فرمول

$$\tau = \frac{(C - D)}{n(n-1)/2}$$

جایی که:
- **C**: تعداد جفت‌های هماهنگ (concordant)
- **D**: تعداد جفت‌های ناهماهنگ (discordant)
- **n**: تعداد مشاهدات

### مثال عملی

```r
# محاسبه ضریب کندال
cor.test(X, Y, method = "kendall")

# نتیجه
# Kendall's rank correlation tau = 0.4
# p-value = 0.33
```

## مقایسه اسپیرمن و کندال

| ویژگی | اسپیرمن | کندال |
|--------|---------|-------|
| اساس | رتبه‌ها | جفت‌های هماهنگ |
| تفسیر | آسان‌تر | دقیق‌تر |
| توانایی آماری | بالاتر | پایین‌تر |
| حساسیت به outlier | کم | کم‌تر |

## کاربردهای عملی

### ۱. ارزیابی کیفیت رتبه‌بندی
```r
# رتبه‌بندی دو قاضی
judge1 <- c(1, 2, 3, 4, 5)
judge2 <- c(1, 3, 2, 5, 4)

# بررسی توافق
cor.test(judge1, judge2, method = "spearman")
```

### ۲. تحلیل داده‌های ترتیبی
```r
# داده‌های رضایت مشتری
satisfaction <- c(1, 2, 3, 4, 5)  # 1=بسیار ناراضی، 5=بسیار راضی
عملکرد <- c(70, 75, 80, 85, 90)

cor.test(satisfaction, عملکرد, method = "kendall")
```

## نتیجه‌گیری

- **اسپیرمن**: زمانی استفاده کنید که نیاز به سرعت و تفسیر آسان دارید
- **کندال**: زمانی استفاده کنید که دقت بیشتر مهم است
