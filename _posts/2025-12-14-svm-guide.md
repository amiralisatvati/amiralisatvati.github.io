## مقدمه

ماشین‌های بردار پشتیبان (Support Vector Machines - SVM) یکی از قوی‌ترین الگوریتم‌های یادگیری نظارت‌شده برای مسائل طبقه‌بندی و رگرسیون است.

## اصول بنیادی SVM

### ۱. مفهوم Hyperplane

SVM تلاش می‌کند یک صفحه (hyperplane) بیدا کند که داده‌های دو کلاس را با بیشترین حاشیه (margin) از یکدیگر جدا کند.

**معادله Hyperplane:**
```
w·x + b = 0
```

جایی که:
- **w**: بردار وزن (normal vector)
- **x**: نقطه داده
- **b**: عرض از مبدا (bias)

### ۲. تابع تصمیم‌گیری

```
f(x) = sign(w·φ(x) + b)
```

## انواع SVM

### Linear SVM
برای داده‌های خطی‌جداپذیر

### Non-Linear SVM
با استفاده از توابع kernel برای داده‌های غیرخطی

## توابع Kernel محبوب

| Kernel | فرمول | کاربرد |
|--------|-------|--------|
| Linear | k(x,y) = x·y | داده‌های خطی |
| RBF | k(x,y) = exp(-γ‖x-y‖²) | داده‌های غیرخطی پیچیده |
| Polynomial | k(x,y) = (x·y + c)^d | داده‌های چند‌جمله‌ای |
| Sigmoid | k(x,y) = tanh(αx·y + c) | شبکه‌های عصبی |

## مثال عملی در R

```r
library(e1071)
library(caret)

# داده‌های آموزشی
data(iris)
set.seed(123)
index <- createDataPartition(iris$Species, p = 0.8, list = FALSE)
train_data <- iris[index, ]
test_data <- iris[-index, ]

# آموزش مدل SVM
svm_model <- svm(
  Species ~ .,
  data = train_data,
  kernel = "radial",
  cost = 1,
  gamma = 0.1
)

# پیش‌بینی
predictions <- predict(svm_model, test_data)

# ارزیابی
accuracy <- mean(predictions == test_data$Species)
print(paste("دقت مدل:", round(accuracy, 4)))
```

## مزایا و معایب

### ✅ مزایا
- عملکرد بالا برای داده‌های ابعادی بالا
- انعطاف‌پذیری با توابع kernel مختلف
- کارآمد برای داده‌های کم

### ❌ معایب
- زمان محاسباتی بالا برای داده‌های بزرگ
- تنظیم پارامترها نیاز به تجربه دارد
- تفسیر پذیری پایین

## نتیجه‌گیری

SVM یک ابزار قدرتمند برای مسائل طبقه‌بندی است، اما نیاز به توجه به انتخاب kernel و پارامترها دارد.
