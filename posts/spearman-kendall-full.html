خلاصه جامع ضرایب همبستگی رتبه‌ای

ضریب همبستگی اسپیرمن (Spearman's Rank Correlation)

تعریف و کاربرد

ضریب همبستگی رتبه‌ای اسپیرمن یک آزمون ناپارامتری برای بررسی رابطه بین دو متغیر بر اساس رتبه‌ها است و زمانی استفاده می‌شود که:

  • حداقل یکی از متغیرها رتبه‌ای (Ordinal) باشد.
  • داده‌ها دارای داده‌های پرت (Outliers) باشند.
  • توزیع داده‌ها نرمال نباشد یا شکل آن نامشخص باشد.

فرمول محاسبه ضریب اسپیرمن

فرمول استاندارد ضریب اسپیرمن به صورت زیر است:

Rs = 1 - ( 6 Σ Di2 / ( n ( n2 - 1 ) ) )

که در آن:

  • Di = Ri - Si (اختلاف بین رتبه‌های دو متغیر برای مشاهده i)
  • n: تعداد نمونه‌ها
  • Ri: رتبه متغیر اول برای مشاهده i
  • Si: رتبه متغیر دوم برای مشاهده i

مراحل محاسبه اسپیرمن

  1. به هر مقدار از متغیر اول و دوم رتبه اختصاص دهید.
  2. برای هر زوج داده، اختلاف رتبه‌ها را محاسبه کنید: Di = Ri - Si.
  3. مجذور اختلاف‌ها را حساب کنید و آن‌ها را جمع کنید: Σ Di2.
  4. مقادیر n و ΣDi2 را در فرمول قرار دهید و Rs را به‌دست آورید.

مثال حل شده ۱: نمره زبان و ریاضی (9 دانشجو)

مسئله: نمرات زبان و ریاضی ۹ دانشجو داده شده است. می‌خواهیم بدانیم آیا بین نمره زبان و ریاضی رابطه‌ای وجود دارد؟ (سطح معنی‌داری α = 0.05)

مرحله ۲: محاسبه ضریب اسپیرمن

با n = 9 و ΣD² = 28 داریم:

Rs = 1 - ( 6 × 28 / ( 9 ( 81 - 1 ) ) )
= 1 - ( 168 / 720 )
≈ 0.767

مرحله ۳: آزمون معنی‌داری (تقریب نرمال)

آماره Z تقریبی برای اسپیرمن:

Z = Rs × √(n - 1)

در این مثال:

Z ≈ 0.767 × √8 ≈ 0.767 × 2.828 ≈ 2.16

چون Z = 2.16 > 1.96، در سطح 0.05 فرض صفر رد می‌شود و نتیجه می‌گیریم بین نمره زبان و ریاضی رابطه معنی‌دار وجود دارد.

ضریب همبستگی کندال (Kendall's Tau)

تعریف و مفهوم

ضریب کندال بر اساس مقایسه زوج‌های مرتب‌شده داده‌ها تعریف می‌شود و میزان توافق یا عدم توافق رتبه‌ها را می‌سنجد.

  • هماهنگ (Concordant): زوج‌هایی که جهت تغییرات آن‌ها یکسان است.
  • ناهماهنگ (Discordant): زوج‌هایی که جهت تغییرات آن‌ها مخالف است.

فرمول محاسبه کندال

τ = ( 4P / ( n ( n - 1 ) ) ) - 1

که در آن:

  • P: تعداد زوج‌های هماهنگ
  • n: تعداد مشاهدات

روش محاسبه P

  1. رتبه‌های یکی از متغیرها را از 1 تا n مرتب کنید.
  2. ردیف دوم را بر اساس همان ترتیب بنویسید.
  3. برای هر عدد، تعداد اعداد بزرگ‌تر سمت راست را بشمارید.
  4. مجموع این تعدادها = P

P = 6 + 6 + 4 + 3 + 3 + 2 + 0 + 0 = 24

مرحله ۳: محاسبه ضریب کندال

با n = 8 و P = 24 داریم:

τ = ( 4 × 24 / ( 8 × 7 ) ) - 1
= 96 / 56 - 1
≈ 1.71 - 1
= 0.71

مرحله ۴: آزمون معنی‌داری

واریانس تقریبی:

Var( τ ) = 2 ( 2n + 5 ) / ( 9 n ( n - 1 ) )
= 42 / 504 ≈ 0.083

آماره Z:

Z = τ / √Var( τ ) = 0.71 / 0.288 ≈ 2.46

چون Z = 2.46 > 1.96، فرض صفر رد می‌شود و بین نظرات دو معلم همبستگی معنی‌دار وجود دارد.

نکات کلیدی برای امتحان

تفسیر مقدار ضریب همبستگی

  • نزدیک به +1: همبستگی مثبت قوی
  • نزدیک به 0: نبود رابطه خطی یا یکنوا
  • نزدیک به -1: همبستگی منفی قوی

تفاوت اسپیرمن و کندال

ویژگی اسپیرمن کندال
مبنای محاسبه بر اساس فاصله رتبه‌ها (D و D²) بر اساس تعداد زوج‌های هماهنگ و ناهماهنگ (P و N)
پیچیدگی محاسبات ساده‌تر برای محاسبه دستی زمان‌برتر برای شمارش زوج‌ها
کارایی در نمونه‌های کوچک خوب، اما حساس‌تر به داده‌های پرت اغلب دقیق‌تر و باثبات‌تر در نمونه‌های کوچک
حساسیت به گره‌ها کمتر تنظیم‌شده برای گره‌ها نسخه‌های اصلاح‌شده (Tau-b و Tau-c) برای گره‌ها مناسب‌تر هستند

فرض‌های آزمون همبستگی رتبه‌ای

  • H0: بین دو متغیر رابطه معنی‌داری وجود ندارد (مستقل هستند).
  • H1: بین دو متغیر رابطه معنی‌داری وجود دارد.

نکته مهم درباره معنی‌داری آماری

اگر آماره Z محاسبه‌شده از مقدار بحرانی (مثلاً 1.96 برای α = 0.05 در آزمون دوطرفه) کمتر باشد، فرض صفر رد نمی‌شود، حتی اگر مقدار ضریب همبستگی از نظر عددی بزرگ به‌نظر برسد.
